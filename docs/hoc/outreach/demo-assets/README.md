# HoC-Republic Proof Demo Assets

This directory contains the first source-backed proof package for **HoC-Republic** outreach. The goal is to make the strongest public hook inspectable before broader researcher, community, or high-profile outreach.

## Reproducible commands

| Command | Output |
| --- | --- |
| `pnpm demo:proof` | Regenerates the digital-genome proof JSON and Markdown artifact. |
| `pnpm demo:proof:video` | Regenerates the no-audio MP4 proof-demo draft from local proof artifacts. |

## Key assets

| Asset | Purpose |
| --- | --- |
| `hoc-republic-digital-genome-proof.md` | Human-readable proof artifact summarizing topology, parent genomes, resource gate, and birth event. |
| `hoc-republic-digital-genome-proof.json` | Machine-readable run artifact suitable for benchmarks and later CI comparison. |
| `hoc-republic-proof-flow.png` | Visual flow diagram for README, demo video, and social posts. |
| `hoc-republic-proof-demo-terminal-transcript.txt` | Clean terminal transcript for the video and public proof posts. |
| `hoc-republic-proof-demo-v0.mp4` | First no-audio proof-demo video draft. |
| `hoc-republic-proof-demo-v0-contact-sheet.png` | Review sheet showing the MP4 frame sequence. |

## Claim boundary

This proof package demonstrates a narrow software claim: HoC-Republic can run repository-backed digital-genome mechanics that create parent genomes, evaluate fitness, select parents, apply crossover and mutation, enforce a resource gate, and emit a traceable birth event. It does **not** claim consciousness, personhood, artificial life, AGI, or biological reproduction.

## Next production step

The MP4 is intentionally conservative and no-audio. The next public-ready version should add a recorded terminal segment, a short voiceover or captions, and a final frame linking to the benchmark roadmap and GitHub repository.
