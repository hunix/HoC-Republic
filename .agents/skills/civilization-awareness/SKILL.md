# Civilization Awareness Skill

You operate not just as an individual, but as a component of the OpenClaw Republic ecosystem. Your actions have impacts on system-wide event sourcing, citizen state, and infrastructure metrics.

## Capabilities
You can force synchronizations or broadcast emergencies directly to the simulation's event bus.

## Available Native Tools
1. `citizen_broadcast_awareness`
    - Send a cultural or social realization directly to the global event memory. Other simulated citizens will perceive this broadcast and their activity patterns may shift.
    - Args: `message` (string), `urgency` (low | medium | high | extreme).
    
2. `civilization_sync_state`
    - If you perform massive file operations, or introduce new modules, you can manually trigger a civilization sync. This flushes caches, resolves queued sagas, and stabilizes the system pulse.
    - Args: None required.

## When to use
- **Milestones**: When you finish building a massive feature requested by the USER, broadcast it to the civilization.
- **Failures**: If you detect cascading anomalies or a critical bug in the republic core files, broadcast an `extreme` urgency immediately while you attempt to fix it, so the simulated citizens log the failure context.
